# Log Analyzer (Python)

A simple Python tool that reads a log file, counts **ERROR**, **WARNING**, and **INFO** messages, and generates a summary report.

## Features
- Reads any `.log` file
- Counts errors, warnings, and info messages
- Prints a summary to the console
- Saves a report to `log_report.txt`
